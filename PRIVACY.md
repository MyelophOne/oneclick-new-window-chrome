# Privacy Policy for OneClick New Window Chrome Extension

**Last Updated:** 2026-05-26

## Overview

OneClick New Window ("the Extension") is a lightweight Chrome extension that allows users to open new browser windows with a single click. This privacy policy explains how the Extension handles user data.

## Data Collection

### No Data Collection

The Extension does **NOT**:

- Collect any personal information
- Track browsing history or activity
- Monitor or record user behavior
- Store any data locally or remotely
- Communicate with external servers or APIs
- Use cookies or similar tracking technologies
- Share data with third parties

### Minimal Permissions

The Extension requires only the `tabs` and `storage` permissions, which are necessary to create new browser window. This permission does not grant access to read or modify tab contents, URLs, or any other tab data.

## How the Extension Works

1. **User Interaction**: When a user clicks the extension icon
2. **Window Creation**: The extension uses Chrome's `chrome.windows.create()` API
3. **No Data Storage**: No information about the action is stored or transmitted

## Technical Implementation

- **Local Processing**: All operations occur locally within the user's browser
- **No Backend**: There is no server-side component
- **No Analytics**: No usage statistics or analytics are collected
- **No Telemetry**: No diagnostic or performance data is sent

## Children's Privacy

The Extension does not knowingly collect any information from children under the age of 13. If you believe a child has provided us with personal information, please contact us immediately.

## Changes to This Privacy Policy

We may update this privacy policy from time to time. We will notify users of any changes by updating the "Last Updated" date at the top of this policy.
